Design and behaviour

This component is displayed with a button when users can perform a main action, usually create a new asset. This component can appear without button for informational purpose only in case the users can't perform any action when the table appears. In case the users could perform an action when the regular table view is displayed, like creating a new asset, please insert a button under the texts.
